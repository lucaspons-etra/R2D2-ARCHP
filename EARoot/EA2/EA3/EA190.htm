Basic Path Basic Path
Details:
Steps:
1. TSO generates and stored IGM in Local TSO IGM repository<br/>
2. local TSO IGM repository is accessed by signer to generate signature<br/>
3. Local TSO IGM repository IGM and signature  is sent to shared IGM repository<br/>
4. shared IGM repository is accesed by RSC to gather IGMS<br/>
5. RSC checks IGM integrity with signature<br/>
6. RSC sends IGM to merging tool<br/>
7. merging tool generates CGM from all IGM<br/>
8. merging tool stores CGM in local RSC IGM/CGM repository<br/>
9. local RSC IGM/CGM repository is used by signer to calculate and store signature<br/>
10. local RSC IGM/CGM repository is accessed by RSC to get CGM and signature<br/>
11. RSC sends CGM and signature to all TSO<br/>

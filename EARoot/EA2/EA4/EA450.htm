Scenario Type
Forest and vegetation reconnaissance Basic Path
Details:
Steps:
1. UAV operator flights the UAV over the selected line<br/>
2. UAV captures lidar data UAV operator downloads them<br/>
3. UAV operator uploads data to EMMA-ARGOS<br/>
4. EMMA-ARGOS identifies problems associates to vegetation in the lidar data<br/>
5. EMMA-ARGOS sends the incidents to EMMA-GIMAN<br/>
Exceptions:
1a. Damages detection (at End)
1b. Electric anomalies detection (at End)
Damages detection Alternate
Details:
Steps:
1. UAV operator flights the UAV over the selected infrastructure (towers, etc)<br/>
2. UAV captures images and UAV operator downloads them<br/>
3. UAV operator uploads images to EMMA-ARGOS<br/>
4. EMMA-ARGOS performs asset 3D reconstruction from the set of images<br/>
5. EMMA-ARGOS automativally identifies structural anomalies from the set of images<br/>
6. EMMA-ARGOS presents 3D model and raw images to system operator<br/>
7. System operator manually identifies structural anomalies from the 3D representation and the raw images in EMMA-ARGOS<br/>
8. EMMA-ARGOS sends incidents to EMMA-GIMAN<br/>
Electric anomalies detection Alternate
Details:
Steps:
1. UAV operator flights the UAV over the selected infrastructure (towers, lines, etc)<br/>
2. UAV captures images and UAV operator downloads them<br/>
3. UAV operator uploads images to EMMA-ARGOS<br/>
4. EMMA-ARGOS automativally identifies electrical anomalies from the set of images<br/>
5. EMMA-ARGOS sends incidents to EMMA-GIMAN<br/>

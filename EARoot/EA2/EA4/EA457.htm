Scenario Type
Compliance simulation scenario Basic Path
Details:
Steps:
1. Grid model server puts selected grid model into the PQEL Server <br/>
2. PQEL server perfoms calculations<br/>
3. PQEL server sends results to power quality server <br/>
Exceptions:
1a. Compliance monitoring scenario (at End)
1b. Equivalent grid parameters calculation scenario (at End)
Compliance monitoring scenario Alternate
Details:
Steps:
1. power quality server sends measurements of the emission levels of flickers, higher harmonics and asymmetries to PQEL server<br/>
2. power quality server sends  calculated emission levels of flickers, higher harmonics and asymmetries to PQEL server<br/>
3. PQEL server calculates - compares the measured and calculated emission levels of flickers, higher harmonics and asymmetries <br/>
4. PQEL server upload calculation results to power quality server<br/>
Equivalent grid parameters calculation scenario Alternate
Details:
Steps:
1. grid model server inputs grid mode linto PQEL server<br/>
2. PQEL server calculates parameters such as R/X, Zd, Zi, Zo, Sk <br/>
3. PQEL server saves the results into power quality server<br/>
